Distance Scripting Framework

JavaScript scripting framework for Minecraft. Create custom HUDs, movement enhancements, combat utilities, and more.

Quick Start

Place scripts in: config/DistanceMod/scripts/

Scripts must end with .dscript

Basic Example

// MyScript.dscript
Distance.log("Script loaded!");
Distance.chat("&aHello from Distance!");

Distance.on("tick", function() {
    // Runs every game tick
});

Distance.on("render2d", function() {
    Distance.drawTextShadow("Hello!", 10, 10, 0xFFFFFF);
});

Features

  • Player position, motion, rotation, health
  • Keyboard and mouse input
  • 2D/3D rendering
  • Particle effects and sounds
  • Custom GUI settings
  • Event system (tick, render, attack)
